# Monster Sound Effects

Place monster sound files here (WAV or OGG format):
- goblin.wav - Goblin growl/attack sound
- spider.wav - Spider hiss
- gargoyle.wav - Stone grinding/wing flap

File name must match monster name (lowercase, spaces become hyphens):
- "Goblin" → goblin.wav
- "Fire Demon" → fire-demon.wav
- "Ancient Dragon" → ancient-dragon.wav

These sounds play automatically when monsters attack in combat.
