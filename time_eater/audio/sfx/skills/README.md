# Skill Sound Effects

Place skill sound files here (WAV or OGG format):
- fire-a.wav - Fire spell sound
- heal.wav - Healing spell sound
- ice-a.wav - Ice spell sound
- poison.wav - Poison effect
- double-strike.wav - Double strike attack
- whirlwind.wav - Whirlwind attack

File name must match skill name (lowercase, spaces become hyphens):
- "Fire A" → fire-a.wav
- "Heal" → heal.wav
- "Ice Storm" → ice-storm.wav

These sounds play automatically when characters use skills in combat.
